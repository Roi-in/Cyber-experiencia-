# Cyber-Experiência 

Este projeto representa meu primeiro contato prático com cibersegurança, utilizando máquinas virtuais, Kali Linux, Metasploitable 2 e a ferramenta Medusa para realizar ataques simulados de força bruta em um ambiente controlado
Após baixar as ISOs e configurar as VMs, pude entrar nesse mundo com curiosidade e entusiasmo. Explorar o Kali Linux e ver a quantidade de ferramentas disponíveis foi uma experiência muito legal. Pesquisar cada uma delas por conta própria me mostrou o tamanho do universo da segurança cibernética.
Durante os testes de penetração utilizando o Kali Linux contra o Metasploitable 2, pude entender na prática como funcionam:
- Wordlists
- Ataques de força bruta com o Medusa
- Enumeração de serviços vulneráveis
- Importância de ambientes isolados para testes
Criar meus próprios arquivos de usuários e senhas para realizar os ataques foi uma etapa muito interessante
Essa primeira experiência foi muito motivadora. Entender o processo de pentest na prática traz uma sensação diferente. Pretendo continuar estudando e explorar outras ferramentas
